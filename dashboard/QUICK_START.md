# 🚀 دليل البدء السريع - نظام إدارة المخبزة

## 📋 الخطوات الأساسية

### 1. تثبيت المكتبات

```bash
npm install
```

### 2. تشغيل المشروع

```bash
npm run dev
```

### 3. فتح المتصفح

```
http://localhost:3000
```
